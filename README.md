# E-commerce Cancellation Analysis

## Overview
Analyzed large-scale e-commerce transaction data (584K+ records, refined to 213K+) to uncover key drivers of order cancellations in Pakistan’s cash-on-delivery (COD) market.

<img src="assets/ecommerce-price-insight.png" width="700"/>

## 🔍 Key Insight
Higher-value orders show significantly higher cancellation risk, making them the most critical segment for intervention.

## Files
- 📓 analysis.ipynb — data cleaning, feature engineering, and exploratory analysis  
- 📄 report.pdf — key insights and actionable recommendations  

## Key Insights
- Cancellation rates increase significantly with higher price ranges  
- Certain product categories consistently show higher risk  
- Discount strategies influence cancellation behavior  

## Recommendations
- Introduce confirmation calls for high-risk orders  
- Optimize pricing and discount strategies  
- Introduce installment-based payment options for high-value orders  
- Offer incentives for digital payments to reduce cancellation risk
