# E-commerce Cancellation Analysis

## Overview
Analyzed cancellation behavior in Pakistan’s cash-on-delivery (COD) market to identify key risk drivers and suggest actionable improvements.

## Files
- analysis.ipynb → technical analysis  
- report.pdf → insights and recommendations  

## Key Insights
- Higher cancellations at higher price ranges  
- Certain product categories show higher risk  
- Discounts influence cancellation behavior  

## Recommendation
- Introduce confirmation for high-risk orders  
- Optimize pricing/discount strategies
